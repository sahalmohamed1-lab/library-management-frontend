# Library Management System - Frontend

## Overview

This is the React frontend for the Library Management System. It communicates with the Django REST API to provide a modern user interface.

## Features

- User authentication
- Protected routes
- Dashboard
- Book management
- Author management
- Category management
- Borrow and return books
- Search books
- Filter books

## Technologies

- React
- React Router
- Axios
- Tailwind CSS

## Installation

Clone the repository

```bash
git clone <repository-url>
cd frontend
```

Install dependencies

```bash
npm install
```

Run the application

```bash
npm run dev
```

The frontend runs on

```
http://localhost:5173
```

- Login
